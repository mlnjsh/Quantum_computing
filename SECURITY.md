# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository (e.g., exposed API tokens, malicious code in notebooks), please report it responsibly:

1. **Do not** open a public issue
2. Email the maintainer or use [GitHub's private vulnerability reporting](https://github.com/mlnjsh/Quantum_computing/security/advisories/new)
3. Include steps to reproduce the issue

## API Tokens

This repository should **never** contain real API tokens. All notebooks use placeholder values like `"YOUR_TOKEN"`. If you find a real token committed, please report it immediately.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (main branch) | Yes |
| Older commits | Best effort |
